# 模板语法

- 变量输出：`{$var}`、`{$user.name}`、`{$list[0]}`
- 修饰符：`{$title|escape|lower|upper|default:'未命名'}`
- 条件：`{if $user.name}{elseif $user.age > 18}{else}{/if}`
- 循环：`{foreach from=$list item=it key=i}{$i}: {$it}{/foreach}`
- 引入：`{include file='shared/footer'}`（继承当前作用域变量）
- 插值：`{{ $expr }}`，代码：`{% $x = 1; %}`
