# 完整示例

## 模板 `views/home/index.tpl`

```tpl
{% $title = '示例'; %}
{$title|upper}
{if $user.name}{$user.name|escape}{else}游客{/if}
{foreach from=$list item=it key=i}{$i}: {$it|default:'空'}{/foreach}
{include file='shared/footer'}
```

## 控制器片段

```php
use PhotonixCore\View\View;
return View::display('home/index', 'cached', [
  'user' => ['name' => '张三'],
  'list' => ['A','B'],
]);
```
