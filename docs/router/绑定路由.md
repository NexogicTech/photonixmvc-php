# 绑定路由到控制器

- 控制器标识采用 `Namespace\\Class/method` 字符串形式。
- 可选显式 `path` 参数用于指向类文件，相对于 `app/`；当省略时框架会自动解析并加载类文件。
- 派发时会调用对应方法并处理返回值：支持 `HtmlResponse` 与 `string`。

参数传递：

- 精确路由无参数，直接调用方法。
- 参数化路由会将占位符 `{name}` 提取为具名参数传入方法。
- 方法形参规则：
  - 无参：直接调用。
  - 单参且名为 `params` 或类型 `array`：整体数组传入。
  - 具名形参：按形参名从参数中匹配，缺省使用默认值或 `null`。

示例：

```
Routes::get('/user/{id}', 'App\\User\\Controller\\Profile/show');

namespace App\\User\\Controller;
class Profile {
    public function show(string $id) {
        echo $id;
    }
}
```
