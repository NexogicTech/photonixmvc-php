# 注册路由

- 在 `app/routes/app.php` 中集中注册应用路由，调度前会自动加载该文件。
- 支持的方法：`get/post/put/delete/patch/options/any`。
- 每条路由包含：`url` 与 `controller`，可选 `path`（相对 `app/` 的类文件路径）。
- 省略 `path` 时，框架会自动解析控制器类对应的文件路径并加载。

示例：

```
use PhotonixCore\\Routes;
Routes::get('/', 'App\\Home\\Controller\\Index/index');
Routes::get('/hello/{name}', 'App\\Home\\Controller\\Index/hello');
Routes::post('/api/login', 'App\\Auth\\Controller\\Login/submit');
Routes::any('/health', 'App\\Health\\Controller\\Probe/ping');
```

注意：

- 路径会被标准化处理，`/path` 与 `/path/` 等价。
- `any` 会匹配所有 HTTP 方法，适用于简单页面或诊断路由。
