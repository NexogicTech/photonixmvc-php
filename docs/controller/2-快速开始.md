# 快速开始

```
namespace App\\Home\\Controller;
use App\\baseController;
use PhotonixCore\\Request;

class Index extends baseController
{
    public function hello(Request $request, string $name = 'World'): string
    {
        $from = $request->get('from', 'web');
        return "Hello, $name from $from";
    }
}
```

在 `app/routes/app.php` 添加：

```
use PhotonixCore\\Routes;
Routes::get('/hello/{name}', 'App\\Home\\Controller\\Index/hello');
```
